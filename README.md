# cost-per-real-view(진행 중)

## CPRV(cost per real view)
* 시선추적을 통해 사용자가 광고를 실제로 보고있는지 아닌지 판단(초록색 점이 사용자의 시선)
* 15초 타이머를 통해 사용자의 시선이 광고에 얼마나 머물렀는지 측정 후 리워드를 할 것인지 측정 (타이머 구현 성공, 시선 머무른 곳 측정은 개발 중)

## 개발 기간
* 2020/8 - 현재
